# L22203 數據處理技術與工具 - 詳細說明

## 一、批次處理

### 1.1 MapReduce
- **Map階段**：映射操作
- **Reduce階段**：歸約操作
- **應用**：大規模數據處理

### 1.2 Apache Spark
- **RDD**：彈性分散式數據集
- **DataFrame**：結構化數據處理
- **Spark SQL**：SQL查詢
- **Spark Streaming**：串流處理

### 1.3 其他工具
- **Apache Flink**：流處理
- **Hadoop生態系統**：分散式處理
