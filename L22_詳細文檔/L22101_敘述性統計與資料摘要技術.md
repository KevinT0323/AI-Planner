# L22101 敘述性統計與資料摘要技術 - 詳細說明

## 一、核心概念與定義

### 1.1 敘述性統計定義
敘述性統計(Descriptive Statistics)是用於描述和總結數據特徵的統計方法，通過各種統計量來呈現數據的分佈、集中趨勢、離散程度等基本特徵。

### 1.2 敘述性統計的目的
- **數據理解**：快速了解數據的基本特徵
- **數據摘要**：用少數指標概括大量數據
- **數據探索**：發現數據中的模式和異常
- **決策支持**：為後續分析提供基礎

### 1.3 敘述性統計 vs 推論統計
- **敘述性統計**：
  - 描述樣本數據
  - 不進行推論
  - 圖表和統計量
- **推論統計**：
  - 從樣本推論母體
  - 假設檢定
  - 區間估計

## 二、集中趨勢量數

### 2.1 平均數(Mean)

#### 2.1.1 算術平均數(Arithmetic Mean)
- **定義**：所有數值的總和除以數值個數
- **公式**：
  - 樣本平均數：x̄ = Σx / n
  - 母體平均數：μ = Σx / N
- **優點**：
  - 考慮所有數值
  - 數學性質良好
  - 易於計算
- **缺點**：
  - 受極端值影響大
  - 不適用於類別數據
- **適用情況**：
  - 數值型數據
  - 數據分佈對稱
  - 無極端值

#### 2.1.2 加權平均數(Weighted Mean)
- **定義**：考慮權重的平均數
- **公式**：x̄w = Σ(wx) / Σw
- **應用**：
  - 不同重要性的數據
  - 分組數據
  - 指數計算

#### 2.1.3 幾何平均數(Geometric Mean)
- **定義**：n個數值乘積的n次方根
- **公式**：GM = (x₁ × x₂ × ... × xₙ)^(1/n)
- **應用**：
  - 增長率計算
  - 比率數據
  - 投資回報

#### 2.1.4 調和平均數(Harmonic Mean)
- **定義**：數值個數除以各數值倒數的總和
- **公式**：HM = n / Σ(1/x)
- **應用**：
  - 速度計算
  - 效率計算

### 2.2 中位數(Median)

#### 2.2.1 定義與計算
- **定義**：將數據排序後位於中間的數值
- **計算方法**：
  - 奇數個數值：中間值
  - 偶數個數值：中間兩個數值的平均
- **優點**：
  - 不受極端值影響
  - 適用於偏態分佈
  - 穩健性高
- **缺點**：
  - 不考慮所有數值
  - 數學性質較差
- **適用情況**：
  - 偏態分佈
  - 有極端值
  - 序數數據

#### 2.2.2 四分位數(Quartiles)
- **Q1 (第一四分位數)**：25%分位數
- **Q2 (第二四分位數)**：中位數，50%分位數
- **Q3 (第三四分位數)**：75%分位數
- **計算方法**：
  - 排序數據
  - 找到對應位置
  - 插值計算（如需要）

#### 2.2.3 百分位數(Percentiles)
- **定義**：數據中小於該值的比例
- **常用百分位數**：
  - P25：第一四分位數
  - P50：中位數
  - P75：第三四分位數
  - P90、P95、P99：高百分位數

### 2.3 眾數(Mode)

#### 2.3.1 定義與計算
- **定義**：出現頻率最高的數值
- **計算方法**：
  - 計數各數值出現次數
  - 找出最大頻率對應的數值
- **優點**：
  - 適用於類別數據
  - 不受極端值影響
  - 易於理解
- **缺點**：
  - 可能不存在
  - 可能有多個
  - 不適用於連續數據
- **適用情況**：
  - 類別數據
  - 離散數據
  - 找出最常見值

#### 2.3.2 單峰與多峰分佈
- **單峰(Unimodal)**：只有一個眾數
- **雙峰(Bimodal)**：有兩個眾數
- **多峰(Multimodal)**：有多個眾數

## 三、離散程度量數

### 3.1 變異數(Variance)與標準差(Standard Deviation)

#### 3.1.1 變異數
- **樣本變異數**：
  - 公式：s² = Σ(x - x̄)² / (n-1)
  - 分母使用(n-1)是為了無偏估計
- **母體變異數**：
  - 公式：σ² = Σ(x - μ)² / N
  - 分母使用N
- **意義**：衡量數據偏離平均數的程度
- **單位**：原數據單位的平方

#### 3.1.2 標準差
- **定義**：變異數的平方根
- **樣本標準差**：s = √s²
- **母體標準差**：σ = √σ²
- **意義**：
  - 與原數據相同單位
  - 更容易解釋
  - 衡量數據分散程度
- **解釋**：
  - 標準差小：數據集中
  - 標準差大：數據分散

#### 3.1.3 變異係數(Coefficient of Variation, CV)
- **定義**：標準差與平均數的比值
- **公式**：CV = s / x̄ × 100%
- **意義**：
  - 相對離散程度
  - 無單位
  - 可比較不同單位的數據
- **應用**：
  - 比較不同單位的變異性
  - 風險評估

### 3.2 全距(Range)

#### 3.2.1 定義與計算
- **定義**：最大值與最小值的差
- **公式**：R = x_max - x_min
- **優點**：
  - 計算簡單
  - 易於理解
- **缺點**：
  - 只考慮兩個極端值
  - 受極端值影響大
  - 忽略中間數據分佈

#### 3.2.2 四分位距(IQR - Interquartile Range)
- **定義**：第三四分位數與第一四分位數的差
- **公式**：IQR = Q3 - Q1
- **優點**：
  - 不受極端值影響
  - 穩健性高
  - 衡量中間50%數據的分散程度
- **應用**：
  - 異常值檢測
  - 箱線圖繪製

### 3.3 偏態(Skewness)與峰態(Kurtosis)

#### 3.3.1 偏態(Skewness)
- **定義**：衡量數據分佈的不對稱程度
- **偏態係數**：
  - 正偏態(Skewness > 0)：右尾較長，平均數 > 中位數
  - 負偏態(Skewness < 0)：左尾較長，平均數 < 中位數
  - 對稱(Skewness ≈ 0)：平均數 ≈ 中位數
- **計算方法**：
  - 樣本偏態：Sk = [n/(n-1)(n-2)] × Σ[(x - x̄)/s]³
- **意義**：
  - 了解數據分佈形狀
  - 選擇合適的統計方法

#### 3.3.2 峰態(Kurtosis)
- **定義**：衡量數據分佈的尖峭程度
- **峰態係數**：
  - 高峰態(Kurtosis > 3)：分佈較尖峭，尾部較厚
  - 低峰態(Kurtosis < 3)：分佈較平坦，尾部較薄
  - 常態峰態(Kurtosis = 3)：標準常態分佈
- **計算方法**：
  - 樣本峰態：Ku = [n(n+1)/(n-1)(n-2)(n-3)] × Σ[(x - x̄)/s]⁴ - 3(n-1)²/(n-2)(n-3)
- **意義**：
  - 了解尾部風險
  - 金融風險評估

## 四、資料摘要技術

### 4.1 頻率分佈表(Frequency Distribution Table)

#### 4.1.1 離散數據頻率表
- **步驟**：
  1. 列出所有可能值
  2. 計數各值出現次數
  3. 計算相對頻率
  4. 計算累積頻率
- **組成**：
  - 數值
  - 頻率(Frequency)
  - 相對頻率(Relative Frequency)
  - 累積頻率(Cumulative Frequency)

#### 4.1.2 連續數據分組頻率表
- **步驟**：
  1. 決定組數
  2. 計算組距
  3. 確定組界
  4. 計數各組頻率
- **組數選擇**：
  - Sturges公式：k = 1 + 3.322 log₁₀(n)
  - 經驗法則：5-20組
- **組距計算**：組距 = (最大值 - 最小值) / 組數

### 4.2 直方圖(Histogram)

#### 4.2.1 定義與繪製
- **定義**：用長條圖表示頻率分佈
- **繪製步驟**：
  1. 確定組距和組界
  2. 計算各組頻率
  3. 繪製長條圖
- **特點**：
  - 連續數據
  - 長條相連
  - 面積代表頻率

#### 4.2.2 解讀直方圖
- **分佈形狀**：
  - 對稱分佈
  - 偏態分佈
  - 多峰分佈
- **異常值識別**：遠離主要分佈的數據點

### 4.3 箱線圖(Box Plot)

#### 4.3.1 定義與組成
- **組成元素**：
  - 箱體：Q1到Q3
  - 中位數線：Q2
  - 鬚線(Whiskers)：延伸到1.5×IQR
  - 異常值：超出1.5×IQR的點
- **繪製步驟**：
  1. 計算Q1、Q2、Q3
  2. 計算IQR
  3. 確定上下界
  4. 繪製箱線圖

#### 4.3.2 解讀箱線圖
- **中位數位置**：了解數據中心
- **箱體長度**：了解IQR，數據分散程度
- **異常值**：識別極端值
- **對稱性**：中位數在箱體中央表示對稱

### 4.4 莖葉圖(Stem-and-Leaf Plot)

#### 4.4.1 定義與繪製
- **定義**：保留原始數據的圖形表示
- **組成**：
  - 莖(Stem)：數據的主要部分
  - 葉(Leaf)：數據的次要部分
- **優點**：
  - 保留原始數據
  - 顯示分佈形狀
  - 易於繪製

### 4.5 五數摘要(Five-Number Summary)

#### 4.5.1 定義與組成
- **組成**：
  1. 最小值(Minimum)
  2. 第一四分位數(Q1)
  3. 中位數(Median, Q2)
  4. 第三四分位數(Q3)
  5. 最大值(Maximum)
- **用途**：
  - 快速了解數據分佈
  - 繪製箱線圖
  - 數據摘要
