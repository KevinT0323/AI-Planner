# L22102 機率分佈與資料分佈模型 - 詳細說明

## 一、核心概念與定義

### 1.1 機率基礎
機率是描述不確定事件發生可能性的數學工具，是統計推論和機器學習的基礎。

### 1.2 隨機變數
- **離散隨機變數**：取值可數的變數
- **連續隨機變數**：取值連續的變數

### 1.3 機率分佈
機率分佈描述隨機變數取值的機率規律。

## 二、機率基礎

### 2.1 基本概念

#### 2.1.1 樣本空間與事件
- **樣本空間(Sample Space)**：所有可能結果的集合
- **事件(Event)**：樣本空間的子集
- **基本事件**：單一結果的事件
- **複合事件**：多個結果的事件

#### 2.1.2 機率公理
- **非負性**：P(A) ≥ 0
- **規範性**：P(Ω) = 1
- **可加性**：互斥事件的機率可加

#### 2.1.3 條件機率
- **定義**：P(A|B) = P(A∩B) / P(B)
- **意義**：在B發生的條件下A發生的機率
- **應用**：貝氏定理、獨立性判斷

#### 2.1.4 貝氏定理(Bayes' Theorem)
- **公式**：P(A|B) = P(B|A) × P(A) / P(B)
- **應用**：
  - 分類問題
  - 診斷測試
  - 機器學習

### 2.2 隨機變數

#### 2.2.1 離散隨機變數
- **定義**：取值可數的隨機變數
- **機率質量函數(PMF)**：P(X = x)
- **性質**：
  - ΣP(X = x) = 1
  - P(X = x) ≥ 0

#### 2.2.2 連續隨機變數
- **定義**：取值連續的隨機變數
- **機率密度函數(PDF)**：f(x)
- **性質**：
  - ∫f(x)dx = 1
  - f(x) ≥ 0
  - P(a ≤ X ≤ b) = ∫[a,b]f(x)dx

#### 2.2.3 累積分佈函數(CDF)
- **定義**：F(x) = P(X ≤ x)
- **離散**：F(x) = ΣP(X = k)，k ≤ x
- **連續**：F(x) = ∫[-∞,x]f(t)dt
- **性質**：
  - 單調遞增
  - F(-∞) = 0，F(∞) = 1

## 三、常見機率分佈

### 3.1 離散分佈

#### 3.1.1 二項分佈(Binomial Distribution)
- **參數**：n（試驗次數）、p（成功機率）
- **PMF**：P(X = k) = C(n,k) × p^k × (1-p)^(n-k)
- **期望值**：E(X) = np
- **變異數**：Var(X) = np(1-p)
- **應用**：n次獨立試驗中成功次數

#### 3.1.2 泊松分佈(Poisson Distribution)
- **參數**：λ（平均發生次數）
- **PMF**：P(X = k) = (λ^k × e^(-λ)) / k!
- **期望值**：E(X) = λ
- **變異數**：Var(X) = λ
- **應用**：單位時間內事件發生次數

#### 3.1.3 幾何分佈(Geometric Distribution)
- **參數**：p（成功機率）
- **PMF**：P(X = k) = (1-p)^(k-1) × p
- **期望值**：E(X) = 1/p
- **應用**：首次成功所需的試驗次數

#### 3.1.4 超幾何分佈(Hypergeometric Distribution)
- **參數**：N（總數）、K（成功數）、n（抽取數）
- **PMF**：P(X = k) = C(K,k) × C(N-K,n-k) / C(N,n)
- **應用**：不放回抽樣中的成功次數

### 3.2 連續分佈

#### 3.2.1 常態分佈(Normal Distribution)
- **參數**：μ（均值）、σ²（變異數）
- **PDF**：f(x) = (1/(σ√(2π))) × e^(-(x-μ)²/(2σ²))
- **性質**：
  - 對稱分佈
  - 鐘形曲線
  - 68-95-99.7規則
- **標準常態分佈**：μ=0，σ=1
- **應用**：許多自然現象

#### 3.2.2 指數分佈(Exponential Distribution)
- **參數**：λ（速率參數）
- **PDF**：f(x) = λe^(-λx)，x ≥ 0
- **期望值**：E(X) = 1/λ
- **變異數**：Var(X) = 1/λ²
- **應用**：等待時間、壽命分佈

#### 3.2.3 均勻分佈(Uniform Distribution)
- **參數**：a（下界）、b（上界）
- **PDF**：f(x) = 1/(b-a)，a ≤ x ≤ b
- **期望值**：E(X) = (a+b)/2
- **變異數**：Var(X) = (b-a)²/12
- **應用**：隨機數生成

#### 3.2.4 t分佈
- **定義**：樣本標準化統計量的分佈
- **應用**：小樣本推論
- **性質**：對稱、尾部較厚

#### 3.2.5 卡方分佈(Chi-square Distribution)
- **定義**：標準常態變數平方和的分佈
- **參數**：自由度k
- **應用**：變異數檢定、適合度檢定

#### 3.2.6 F分佈
- **定義**：兩個卡方分佈比值的分佈
- **參數**：兩個自由度
- **應用**：變異數比檢定、ANOVA

## 四、資料分佈模型

### 4.1 參數估計方法

#### 4.1.1 點估計
- **定義**：用單一數值估計參數
- **估計量**：樣本統計量
- **性質**：
  - 無偏性：E(θ̂) = θ
  - 有效性：變異數小
  - 一致性：樣本增大趨近真值

#### 4.1.2 最大似然估計(MLE)
- **原理**：選擇使觀察數據機率最大的參數
- **方法**：
  - 建立似然函數
  - 對數轉換
  - 求導數為零
- **優點**：漸近有效、漸近常態

#### 4.1.3 動差估計法(Method of Moments)
- **原理**：樣本動差等於母體動差
- **方法**：
  - 計算樣本動差
  - 設定等於母體動差
  - 求解參數
- **優點**：簡單直觀

### 4.2 分佈擬合檢驗

#### 4.2.1 卡方適合度檢定
- **目的**：檢驗數據是否符合某分佈
- **步驟**：
  1. 假設分佈
  2. 計算期望頻率
  3. 計算卡方統計量
  4. 與臨界值比較

#### 4.2.2 Kolmogorov-Smirnov檢定
- **目的**：檢驗連續分佈
- **方法**：比較經驗分佈函數與理論分佈函數

#### 4.2.3 常態性檢定
- **Shapiro-Wilk檢定**：小樣本
- **Anderson-Darling檢定**：一般樣本
- **視覺化方法**：Q-Q圖、直方圖

### 4.3 分佈選擇策略
- **數據特徵**：離散/連續、範圍
- **理論依據**：問題背景
- **統計檢定**：適合度檢定
- **視覺化檢查**：圖形比較
