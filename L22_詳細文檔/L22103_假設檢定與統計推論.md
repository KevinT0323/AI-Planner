# L22103 假設檢定與統計推論 - 詳細說明

## 一、核心概念與定義

### 1.1 統計推論定義
統計推論是從樣本推論母體特徵的過程，包括點估計、區間估計和假設檢定。

### 1.2 假設檢定定義
假設檢定是根據樣本數據判斷母體參數是否符合某種假設的統計方法。

### 1.3 重要性
- **科學決策**：基於數據做決策
- **品質控制**：檢驗產品品質
- **研究驗證**：驗證研究假設

---

## 二、假設檢定基礎

### 2.1 基本概念

#### 2.1.1 虛無假設(H₀)與對立假設(H₁)
- **虛無假設(H₀)**：
  - 待檢驗的假設
  - 通常表示"無差異"或"無效果"
  - 例：H₀: μ = μ₀
- **對立假設(H₁)**：
  - 與虛無假設對立
  - 通常表示"有差異"或"有效果"
  - 例：H₁: μ ≠ μ₀（雙尾）或 μ > μ₀（單尾）

#### 2.1.2 第一型錯誤(Type I Error)與第二型錯誤(Type II Error)
- **第一型錯誤(α)**：
  - 定義：H₀為真但拒絕H₀
  - 機率：α = P(拒絕H₀ | H₀為真)
  - 意義：假陽性
- **第二型錯誤(β)**：
  - 定義：H₀為假但接受H₀
  - 機率：β = P(接受H₀ | H₀為假)
  - 意義：假陰性
- **檢定力(Power)**：
  - 定義：1 - β
  - 意義：正確拒絕H₀的機率

#### 2.1.3 顯著水準(Significance Level, α)
- **定義**：允許的第一型錯誤機率
- **常用值**：0.05、0.01、0.10
- **意義**：檢定的嚴格程度

#### 2.1.4 檢定力(Power, 1-β)
- **定義**：正確拒絕H₀的機率
- **影響因素**：
  - 樣本大小：樣本越大，檢定力越高
  - 效果量：效果越大，檢定力越高
  - 顯著水準：α越大，檢定力越高
- **目標**：通常希望 ≥ 0.80

### 2.2 檢定流程
1. **假設設定**：設定H₀和H₁
2. **選擇檢定統計量**：根據數據類型選擇
3. **決定拒絕域**：根據α和H₁決定
4. **計算p值**：計算觀察值或更極端的機率
5. **做出決策**：比較p值與α

## 三、常見檢定方法

### 3.1 單樣本檢定

#### 3.1.1 單樣本t檢定
- **適用**：樣本平均數檢定，母體標準差未知
- **假設**：
  - H₀: μ = μ₀
  - H₁: μ ≠ μ₀（雙尾）或 μ > μ₀（單尾）
- **檢定統計量**：t = (x̄ - μ₀) / (s/√n)
- **分佈**：t分佈，自由度n-1
- **條件**：樣本來自常態分佈或樣本大(n≥30)

#### 3.1.2 單樣本比例檢定
- **適用**：樣本比例檢定
- **假設**：
  - H₀: p = p₀
  - H₁: p ≠ p₀
- **檢定統計量**：z = (p̂ - p₀) / √(p₀(1-p₀)/n)
- **分佈**：標準常態分佈
- **條件**：np₀ ≥ 5，n(1-p₀) ≥ 5

#### 3.1.3 單樣本變異數檢定
- **適用**：樣本變異數檢定
- **假設**：
  - H₀: σ² = σ₀²
  - H₁: σ² ≠ σ₀²
- **檢定統計量**：χ² = (n-1)s² / σ₀²
- **分佈**：卡方分佈，自由度n-1

### 3.2 雙樣本檢定

#### 3.2.1 獨立樣本t檢定
- **適用**：兩個獨立樣本平均數比較
- **假設**：
  - H₀: μ₁ = μ₂
  - H₁: μ₁ ≠ μ₂
- **檢定統計量**：
  - 變異數相等：t = (x̄₁ - x̄₂) / sp√(1/n₁ + 1/n₂)
  - 變異數不等：Welch's t檢定
- **條件**：兩樣本獨立、常態分佈

#### 3.2.2 配對樣本t檢定
- **適用**：配對樣本平均數比較
- **假設**：
  - H₀: μd = 0
  - H₁: μd ≠ 0
- **檢定統計量**：t = d̄ / (sd/√n)
- **優點**：控制個體差異

#### 3.2.3 兩樣本比例檢定
- **適用**：兩個樣本比例比較
- **假設**：
  - H₀: p₁ = p₂
  - H₁: p₁ ≠ p₂
- **檢定統計量**：z = (p̂₁ - p̂₂) / √(p̂(1-p̂)(1/n₁ + 1/n₂))

#### 3.2.4 F檢定(變異數比檢定)
- **適用**：兩個樣本變異數比較
- **假設**：
  - H₀: σ₁² = σ₂²
  - H₁: σ₁² ≠ σ₂²
- **檢定統計量**：F = s₁² / s₂²
- **分佈**：F分佈

### 3.3 多樣本檢定

#### 3.3.1 單因子變異數分析(ANOVA)
- **適用**：多個樣本平均數比較
- **假設**：
  - H₀: μ₁ = μ₂ = ... = μk
  - H₁: 至少一個不相等
- **檢定統計量**：F = MSB / MSW
- **條件**：
  - 獨立樣本
  - 常態分佈
  - 變異數齊性
- **事後檢定**：Tukey HSD、Bonferroni

#### 3.3.2 卡方檢定(Chi-square Test)
- **適合度檢定**：
  - 檢驗觀察頻率是否符合理論分佈
  - 檢定統計量：χ² = Σ(Oi - Ei)² / Ei
- **獨立性檢定**：
  - 檢驗兩個變數是否獨立
  - 列聯表分析

#### 3.3.3 無母數檢定(Non-parametric Tests)
- **Wilcoxon符號秩檢定**：配對樣本
- **Mann-Whitney U檢定**：兩獨立樣本
- **Kruskal-Wallis檢定**：多獨立樣本
- **優點**：不要求常態分佈

## 四、統計推論

### 4.1 區間估計

#### 4.1.1 信賴區間(Confidence Interval)
- **定義**：包含母體參數的區間
- **信賴水準**：1 - α（通常95%）
- **意義**：重複抽樣，100(1-α)%的區間包含真值

#### 4.1.2 信賴水準的意義
- **95%信賴區間**：重複抽樣，95%的區間包含真值
- **誤解**：不是參數在區間內的機率為95%
- **正確理解**：區間建構方法的可靠性

#### 4.1.3 平均數信賴區間
- **母體標準差已知**：
  - x̄ ± z_(α/2) × (σ/√n)
- **母體標準差未知**：
  - x̄ ± t_(α/2,n-1) × (s/√n)

#### 4.1.4 比例信賴區間
- **公式**：p̂ ± z_(α/2) × √(p̂(1-p̂)/n)
- **條件**：np̂ ≥ 5，n(1-p̂) ≥ 5

### 4.2 樣本大小決定

#### 4.2.1 樣本大小計算
- **平均數檢定**：
  - n = (z_(α/2) + z_β)² × σ² / (μ₁ - μ₀)²
- **比例檢定**：
  - n = (z_(α/2) + z_β)² × p(1-p) / (p₁ - p₀)²

#### 4.2.2 檢定力分析
- **目的**：確定樣本大小以達到所需檢定力
- **輸入**：
  - 效果量
  - 顯著水準
  - 檢定力目標
- **輸出**：所需樣本大小

#### 4.2.3 效果量(Effect Size)
- **定義**：標準化的差異大小
- **Cohen's d**：d = (μ₁ - μ₂) / σ
- **解釋**：
  - 小：0.2
  - 中：0.5
  - 大：0.8

## 五、p值解釋

### 5.1 p值定義
- **定義**：在H₀為真的條件下，觀察到檢定統計量等於或更極端的機率
- **計算**：P(檢定統計量 ≥ 觀察值 | H₀為真)

### 5.2 p值解釋
- **p < α**：拒絕H₀，結果顯著
- **p ≥ α**：不拒絕H₀，結果不顯著
- **常見誤解**：
  - p值不是H₀為真的機率
  - p值不是效果大小
  - p值不是實務重要性

### 5.3 p值與決策
- **p < 0.001**：極強證據反對H₀
- **p < 0.01**：強證據反對H₀
- **p < 0.05**：證據反對H₀
- **p ≥ 0.05**：證據不足反對H₀
