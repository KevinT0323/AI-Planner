## 第一部分：核心概念與定義 (5題)

### 1. 數據儲存的主要目的是？
A. 數據加密
B. 數據可視化  
C. 將數據持久化保存到儲存介質中，確保安全性與可訪問性  
D. 刪除數據  
<details>
<summary>顯示答案</summary>

**正確答案：C**  
**解析：** 數據儲存的核心在於持久化保存數據以便後續使用。

</details>

### 2. 數據管理的定義包含對數據生命週期的哪些操作？
A. 僅備份  
B. 僅刪除  
C. 規劃、組織、控制和監督  
D. 僅加密
<details>
<summary>顯示答案</summary>

**正確答案：C**  
**解析：** 數據管理是對數據全生命週期的全面管控過程。

</details>

### 3. 持久化 (Persistence) 在數據儲存中的意義是？
A. 數據在斷電後仍然存在  
B. 數據無法被刪除
C. 數據只存在記憶體中  
D. 數據永遠不會改變  
<details>
<summary>顯示答案</summary>

**正確答案：A**  
**解析：** 持久化是指數據寫入非易失性存儲介質，不會因斷電而丟失。

</details>

### 4. 下列何者不屬於數據生命週期的階段？
A. 使用  
B. 忽視  
C. 歸檔
D. 創建  
<details>
<summary>顯示答案</summary>

**正確答案：B**  
**解析：** 數據生命週期包括創建、使用、歸檔、刪除等，不包括"忽視"。

</details>

### 5. 數據的可訪問性是指？
A. 任何人都可隨意修改數據  
B. 授權用戶能夠方便、及時地獲取數據  
C. 數據必須公開在網路上  
D. 數據不能被加密
<details>
<summary>顯示答案</summary>

**正確答案：B**  
**解析：** 可訪問性強調在需要時能夠存取數據。

</details>

## 第二部分：關聯式資料庫 (RDBMS) (5題)

### 6. RDBMS 的全稱是？
A. Random Data Base System  
B. Rapid Data Base System
C. Relational Database Management System  
D. Real Database Management System  
<details>
<summary>顯示答案</summary>

**正確答案：C**  
**解析：** RDBMS 指關聯式資料庫管理系統。

</details>

### 7. ACID 特性中的 "A" 代表？
A. Availability (可用性)  
B. Authorization (授權)
C. Atomicity (原子性)  
D. Accuracy (準確性)  
<details>
<summary>顯示答案</summary>

**正確答案：C**  
**解析：** Atomicity 確保交易要麼全部完成，要麼全部取消。

</details>

### 8. ACID 特性中的 "I" 代表？
A. Intelligence (智能)  
B. Integrity (完整性)  
C. Isolation (隔離性)  
D. Identity (身份)
<details>
<summary>顯示答案</summary>

**正確答案：C**  
**解析：** Isolation 確保並發交易互不干擾。

</details>

### 9. 關聯式資料庫使用什麼語言進行查詢？
A. Python  
B. Java
C. HTML  
D. SQL (Structured Query Language)  
<details>
<summary>顯示答案</summary>

**正確答案：D**  
**解析：** SQL 是操作 RDBMS 的標準語言。

</details>

### 10. 下列哪一個不是 RDBMS 的代表產品？
A. Oracle  
B. MongoDB
C. PostgreSQL  
D. MySQL  
<details>
<summary>顯示答案</summary>

**正確答案：B**  
**解析：** MongoDB 是 NoSQL 資料庫。

</details>

## 第三部分：NoSQL 資料庫 (5題)

### 11. NoSQL 的含義通常被解釋為？
A. New SQL
B. Not Only SQL (不僅僅是SQL)  
C. No SQL (不使用SQL)  
D. Never SQL  
<details>
<summary>顯示答案</summary>

**正確答案：B**  
**解析：** NoSQL 強調它不只是SQL，能處理多種數據模型。

</details>

### 12. 下列何者是文件資料庫 (Document Store) 的代表？
A. Cassandra  
B. Neo4j
C. Redis  
D. MongoDB  
<details>
<summary>顯示答案</summary>

**正確答案：D**  
**解析：** MongoDB 是最流行的文件型資料庫。

</details>

### 13. Redis 屬於哪一類 NoSQL 資料庫？
A. 圖形資料庫  
B. 列式資料庫
C. 文件資料庫  
D. 鍵值資料庫 (Key-Value Store)  
<details>
<summary>顯示答案</summary>

**正確答案：D**  
**解析：** Redis 是高性能的鍵值存儲系統。

</details>

### 14. 圖形資料庫 (Graph Database) 最適合處理？
A. 文件歸檔
B. 簡單的日誌數據  
C. 複雜的關聯網絡 (如社交關係)  
D. 圖片存儲  
<details>
<summary>顯示答案</summary>

**正確答案：C**  
**解析：** 圖形資料庫擅長處理節點與邊的關係，如社交網絡。

</details>

### 15. 列式資料庫 (Column-family Store) 的代表是？
A. Cassandra  
B. MySQL  
C. Redis  
D. SQLite
<details>
<summary>顯示答案</summary>

**正確答案：A**  
**解析：** Cassandra 和 HBase 是列式資料庫的代表。

</details>

## 第四部分：分散式儲存與數據倉儲 (5題)

### 16. HDFS 的全稱是？
A. High Data File System  
B. Hadoop Distributed File System  
C. Huge Disk File System  
D. Hard Drive File System
<details>
<summary>顯示答案</summary>

**正確答案：B**  
**解析：** HDFS 是 Hadoop 的核心組件。

</details>

### 17. 資料分片 (Sharding) 的最主要目的是？
A. 備份數據  
B. 壓縮數據
C. 加密數據  
D. 水平分割數據以分散負載與存儲壓力  
<details>
<summary>顯示答案</summary>

**正確答案：D**  
**解析：** Sharding 將數據切分到不同節點，實現水平擴展。

</details>

### 18. 資料複製 (Replication) 的主要好處是？
A. 加快寫入速度  
B. 節省空間  
C. 減少網路流量
D. 提高可用性與容錯能力  
<details>
<summary>顯示答案</summary>

**正確答案：D**  
**解析：** 副本可以在主節點故障時提供服務，提高可用性。

</details>

### 19. ETL 代表哪三個步驟？
A. Edit, Type, Lock  
B. Enter, Test, Log  
C. Encrypt, Transfer, Link
D. Extract, Transform, Load  
<details>
<summary>顯示答案</summary>

**正確答案：D**  
**解析：** 抽取、轉換、載入是數據倉儲的核心流程。

</details>

### 20. 數據湖 (Data Lake) 通常存儲什麼類型的數據？
A. 僅存儲 Excel 文件  
B. 僅存儲圖片
C. 僅存儲清洗後的數據  
D. 原始格式的各種數據 (結構化、半結構化、非結構化)  
<details>
<summary>顯示答案</summary>

**正確答案：D**  
**解析：** 數據湖強調保留數據的原始狀態。

</details>

## 第五部分：數據管理與治理 (5題)

### 21. 數據治理 (Data Governance) 的核心目標是？
A. 開發軟體
B. 購買更多伺服器  
C. 限制數據使用  
D. 確保數據的高品質、安全與合規，並最大化數據價值  
<details>
<summary>顯示答案</summary>

**正確答案：D**  
**解析：** 數據治理是關於數據資產管理的制度與流程。

</details>

### 22. 數據標準 (Data Standards) 的作用是？
A. 強制使用特定軟體  
B. 確保數據在組織內的定義和格式統一  
C. 限制數據量  
D. 增加數據複雜度
<details>
<summary>顯示答案</summary>

**正確答案：B**  
**解析：** 統一標準是數據整合與溝通的基礎。

</details>

### 23. 數據品質管理 (Data Quality Management) 不包括下列哪項？
A. 硬碟維修
B. 準確性驗證  
C. 完整性檢查  
D. 一致性維護  
<details>
<summary>顯示答案</summary>

**正確答案：A**  
**解析：** 硬碟維修屬於硬體維護。

</details>

### 24. 數據生命週期的最後一個階段通常是？
A. 刪除 (或銷毀)
B. 創建  
C. 分享  
D. 使用  
<details>
<summary>顯示答案</summary>

**正確答案：A**  
**解析：** 當數據不再需要且無法規保留要求時，應被安全銷毀。

</details>

### 25. 數據歸檔與備份的區別在於？
A. 備份為了災難恢復，歸檔為了長期保留與合規  
B. 沒有區別  
C. 備份是永久的，歸檔是暫時的  
D. 歸檔比備份快
<details>
<summary>顯示答案</summary>

**正確答案：A**  
**解析：** 備份是為了救急（如誤刪），歸檔是為了查閱歷史。

</details>
